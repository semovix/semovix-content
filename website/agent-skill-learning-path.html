基础认知与概念体系

理解 Agent 的本质，建立核心概念框架

Foundation
🤖

什么是 AI Agent

理解 Agent 与普通 LLM 的本质区别——感知、思考、行动的循环；从 ReAct 框架出发，掌握 Reasoning + Acting 的核心思路。

ReAct LLM基础 Agent定义

🧩

Prompt Engineering for Agent

Agent 的行为高度依赖 Prompt 设计。掌握系统提示词、角色定义、CoT（思维链）、Few-shot 等技巧，为后续工具调用打下基础。

CoT System Prompt Few-shot
学习内容
思维链（Chain-of-Thought）提示技巧
任务分解与逐步推理设计
自我反思（Self-Reflection）提示模式
⏱ 约 1 周
🗺️

Agent 架构全景

系统了解主流 Agent 架构：单一 Agent、管道式 Agent、多智能体系统；理解 Planning、Memory、Tools、Action 四大核心模块的职责与协作关系。

Planning Memory Tools Action
学习内容
四大模块的职责划分与接口设计
常见架构模式对比（Linear / DAG / Loop）
LangChain / AutoGen 架构调研

02

核心能力模块开发

动手实现 Agent 的工具使用、规划与记忆系统

Core Skills
🔧

工具调用（Tool Use / Function Calling）

掌握让 LLM 调用外部工具的核心机制，包括工具定义格式、参数解析、结果注入与错误处理，实现搜索、代码执行、数据库查询等场景。

Function Call JSON Schema Tool Registry
实战项目
构建带搜索能力的 Q&A Agent
实现代码执行沙箱工具调用
工具错误重试与降级策略
⏱ 约 2 周
🧠

记忆系统设计

理解并实现 Agent 的四种记忆类型：感知记忆（Sensory）、短期工作记忆（Working）、情节记忆（Episodic）、语义长期记忆（Semantic）。

Vector DB RAG Context Window
实战项目
基于 Chroma / Qdrant 的长期记忆实现
记忆检索与上下文注入策略
记忆遗忘与优先级管理机制
⏱ 约 2 周
📋

任务规划（Planning）

实现 Agent 将复杂任务分解为子步骤的规划能力，对比 Chain-of-Thought、Tree of Thoughts、Plan-and-Solve 等策略，处理动态重规划。

ToT MCTS ReWOO
实战项目
实现 Plan-Execute-Reflect 循环
动态任务图与依赖解析
规划失败检测与自动修复
⏱ 约 1.5 周
🔄

状态管理与执行控制

设计 Agent 的状态机（State Machine），掌握同步/异步执行、超时中断、幂等重试、事件驱动等工程实践，确保 Agent 稳定可靠运行。

状态机 异步执行 幂等性
实战项目
基于 LangGraph 的状态图构建
异步工具调用与并发控制
Agent 执行轨迹日志与回放
⏱ 约 1.5 周

多智能体系统

从单一 Agent 扩展到多 Agent 协作，构建复杂分工体系

Multi-Agent
👥

多智能体协作模式

掌握三种核心多 Agent 模式：协作（Collaboration）、竞争（Competition）、辩论（Debate），理解 Supervisor-Worker 与 Peer-to-Peer 架构的适用场景。

📡

Agent 通信与消息协议

设计 Agent 间的通信机制，掌握消息队列、事件总线、共享黑板等模式；实现结构化的 Agent-to-Agent 通信协议，处理消息路由与冲突解决。

消息队列 事件驱动 共享状态

⏱ 约 1.5 周
🛡️

安全、对齐与信任边界

Agent 的安全是生产落地的关键。学习 Prompt Injection 防御、工具权限管控、输出内容过滤、人机协作审批（Human-in-the-Loop）等安全实践。

HITL Guardrails Prompt Injection
实战项目
Prompt Injection 攻击与防御演练
工具调用白名单与权限分级
关键操作人工审批节点设计
⏱ 约 1.5 周
04

生产落地与持续优化

将 Agent 从原型推向可靠的生产系统

Production
📊

评估体系（Evaluation）

建立 Agent 的定量评估框架，区分任务完成率、工具使用准确率、幻觉率、延迟与成本等指标；设计 Benchmark 与自动化测试套件。

AgentBench Tracing Metrics
实战项目

幻觉检测与 Groundedness 评分
端到端 Trace 与可观测性接入

⚡

性能优化与成本控制

降低 Agent 的推理成本与延迟：Token 压缩、动态模型路由（强/弱模型协作）、并行工具调用、结果缓存与 Prompt 复用策略。

Token优化 模型路由 并行化
实战项目
Token 使用分析与压缩优化
强弱模型自适应路由实现
