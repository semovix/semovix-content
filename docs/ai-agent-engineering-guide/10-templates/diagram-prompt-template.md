```text
请生成一张「{图名称}」的高质量技术信息图。

{主题}

目标：
{这张图要帮助读者理解什么}

风格：
- 百科全书式
- 教材讲义式
- 高信息密度
- 专业技术白皮书
- 扁平化图标
- 结构化知识树
- 中文必须准确清晰

必须包含：
{模块列表}

结构要求：
- 顶部：核心标题与一句话定义
- 中部：主要架构 / 流程 / 能力分层
- 底部：关键原则、学习路径或结论

视觉要求：
- 字体清晰，小字可读
- 模块边界明确
- 箭头方向清楚
- 颜色克制，适合打印和培训

比例：
21:9

质量：
8K ultra HD, crisp typography, clean infographic, professional layout
```
